# EcoClean
EcoClean is an AI-powered digital waste management system that detects redundant data across local storage, cloud drives, and email platforms. It uses context, privacy sensitivity, and carbon-impact metrics to recommend safe cleanup actions, reducing storage waste and digital carbon emissions.
## Problem Statement
Digital storage redundancy increases cost, risk, and carbon footprint.

## Solution
EcoClean provides context-aware, privacy-safe, carbon-optimized cleanup.

## System Workflow
1. Data collection
2. Deduplication
3. Privacy analysis
4. Carbon estimation
5. Retention scoring
